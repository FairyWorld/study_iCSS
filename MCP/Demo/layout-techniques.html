1. Flex 自动 margin 居中

在 Flex 布局中，使用 margin: auto 可以轻松实现元素的水平垂直居中，这是最简洁的居中方案。

效果演示：
居中
.g-container { display: flex; height: 200px; } .center-box { margin: auto; width: 100px; height: 100px; }
🔍 原理说明：
在 Flex 格式化上下文中，margin: auto 会自动分配剩余空间。水平方向分配左右 margin，垂直方向分配上下 margin，从而实现完美居中。

2. CSS Grid 布局

CSS Grid 是最强大的二维布局系统，可以同时控制行和列，实现复杂的网格布局。

效果演示：

.grid-layout { display: grid; grid-template-columns: repeat(3, 1fr); gap: 15px; }
🔍 原理说明：
Grid 布局通过 grid-template-columns 定义列，grid-template-rows 定义行，使用 gap 设置间距，实现精确的二维布局控制。

3. 多列等高布局

使用 Flex 布局可以轻松实现多列等高效果，所有列的高度会自动适应最高的那一列。

效果演示：

中间列内容
多行文本
右列
.equal-height { display: flex; gap: 20px; } .column { flex: 1; /* 自动等高 */ }
🔍 原理说明：
Flex 布局默认会拉伸子元素到容器高度，通过 flex: 1 让所有列平均分配宽度，同时自动适应最高列的高度。

.waterfall { column-count: 3; column-gap: 20px; } .waterfall-item { break-inside: avoid; margin-bottom: 20px; }
🔍 原理说明：
column-count 定义列数，内容自动按列分布。break-inside: avoid 防止元素被分割到不同列。

7. 形状布局 (Shape)

使用 shape-outside 创建非矩形布局，文字可以围绕任意形状排列。

效果演示：
文字会围绕圆形形状排列，创造出有趣的布局效果。这种技术常用于杂志风格的网页设计。
.shape-element { shape-outside: circle(50%); float: left; border-radius: 50%; }
🔍 原理说明：
shape-outside 定义元素的形状边界，文字会围绕这个形状排列。支持 circle、polygon、ellipse 等形状。

8. 响应式布局

使用 Grid 的 auto-fit 和 minmax 实现自适应响应式布局。

9. 文本布局技巧

单行居中多行居左的文本布局技巧，结合省略号效果。

效果演示：

.single-line-center { text-align: center; white-space: nowrap; overflow: hidden; text-overflow: ellipsis; } .multi-line-left { text-align: left; display: -webkit-box; -webkit-line-clamp: 2; -webkit-box-orient: vertical; overflow: hidden; }
🔍 原理说明：
单行使用 text-align: center 和 text-overflow: ellipsis，多行使用 -webkit-line-clamp 限制行数并显示省略号。

10. 粘性页脚布局

使用 Flex 布局实现粘性页脚，内容少时页脚在底部，内容多时页脚跟随内容。

效果演示：
页面内容区域
页脚始终在底部
.sticky-footer-container { height: 300px; display: flex; flex-direction: column; } .sticky-footer-content { flex: 1; } .sticky-footer { flex-shrink: 0; }
🔍 原理说明：
使用 flex-direction: column 垂直布局，内容区域 flex: 1 占据剩余空间，页脚 flex-shrink: 0 保持固定高度。
